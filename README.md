# Portfólio — Nicholas Diego de Lira Veloso
Portfólio profissional desenvolvido como landing page para apresentação de projetos, habilidades técnicas e experiência na área de desenvolvimento de software.

## Visão Geral

Este projeto foi desenvolvido com o objetivo de apresentar meu perfil profissional de forma clara, moderna e objetiva, servindo como ponto central para recrutadores e contatos profissionais.

O foco do portfólio é destacar minha atuação como Desenvolvedor Backend, bem como minha capacidade de construir interfaces modernas, performáticas e bem estruturadas no frontend.

## Tecnologias Utilizadas

### Frontend
- React
- TypeScript
- Tailwind CSS
- GSAP (animações sutis)

### Backend & Conceitos
- Java
- Spring Boot
- APIs REST
- Arquitetura em camadas

### Infraestrutura
- AWS (EC2, RDS)
- Git & GitHub

## Principais Características

- Design moderno com foco em experiência do usuário
- Layout totalmente responsivo (mobile-first)
- Animações sutis e acessíveis com GSAP
- Navegação fluida e clara
- Boas práticas de acessibilidade (A11y)
- Estrutura de código organizada e escalável

## Como Executar Localmente

Clone o repositório:

```bash
git clone https://github.com/nickdiegao/Portfolio-Nicholas-Diego.git


Acesse a pasta do projeto:

```bash
cd Portfolio-Nicholas-Diego
```

Instale as depêndencias:

```bash
npm install
```

Incie o projeto:
 
```bash
npm run dev
```
